このパッケージは、"PSP"用 2D 画像データを扱うために必要なコンバータ、
ビューア、ドキュメントが含まれています。

[gimconv ver 1.50]
gimconv は 2D 画像コンバータで、一般的な 2D 画像データを "PSP" 用 
2D 画像データに変換します。

[gimview ver 1.50]
gimview は 2D 画像ビューアで、gimconv で作成した "PSP" 用 2D 画像
データを PC 上で確認することができます。

使用方法
----------------------------------------------------------------------
- gimconv ver1.50

  usage:
    gimconv <入力ファイル(*.bmp,*.tga,*.tm2,*.dds,*.avi,*.gim,*.png)> [オプション]

  options:
    -interact     : 追加オプションをキー入力
    -pictures     : 複数の「ピクチャ」として１つのファイルに結合
    -frames       : 複数の「フレーム」として１つのファイルに結合
    -levels       : 複数の「レベル」として１つのファイルに結合
    -prompt       : 常にキー入力待ちをおこなう
    -warning      : 警告時にキー入力待ちをおこなう
    -error        : エラー時にキー入力待ちをおこなう
    -viewer       : 終了時に gimview.exe を起動する
    -o <filename> : 出力ファイル名を指定する
    -s <w,h>      : 画像データをスケーリングする
    -S            : テキスト形式を出力する
    -P            : 画像データを 2 のべき乗にスケーリングする
    -N            : 通常ピクセル格納形式で出力する
    -F            : 高速ピクセル格納形式で出力する

- gimview ver1.50
 
  usage:
    gimview <入力ファイル(*.gim)> [options]

  options:
    -fg <code>     : foreground color ( hex-code )
    -bg <code>     : background color ( hex-code )
    -nomip         : disable mipmap

  help:
    ESCAPE KEY    : QUIT PROGRAM
    SPACE KEY     : LOAD NEXT MODEL
    BACKSPACE KEY : LOAD PREV MODEL
    RETURN KEY    : RESET POSITION
    TAB KEY       : PAUSE ANIMATION
    1,2,3 .. KEY  : CHANGE CONTROL MODE

    MOUSE-L BUTTON : ROTATE
    MOUSE-R BUTTON : MOVE
    Z,X KEY       : ZOOM
    C,V KEY       : ROLL
    +,- KEY       : RESIZE

    F1 KEY        : TOGGLE BLEND
    F2 KEY        : TOGGLE FUNC
    F3 KEY        : TOGGLE FILTER
    F4 KEY        : TOGGLE WRAP
    F5 KEY        : TOGGLE BG
    F6 KEY        : TOGGLE FG
    F7 KEY        : TOGGLE GRID
    F8 KEY        : TOGGLE BUFFER
    F9 KEY        : SHOW FILE INFO
